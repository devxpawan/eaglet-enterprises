# Business Management System (BMS)

## Features

### 🔐 Authentication & User Management
- Secure login and registration system
- Role-based access control (Admin, Moderator, etc.)
- User profile management with password updates
- User activity logs for tracking actions
- User CRUD (Add, Edit, Update, Delete)

### 👥 Customer Management
- Add, edit, and update customer information
- Customer list with search and filtering
- Toggle customer active/inactive status
- Customer details management

### 📦 Product Management
- Add, edit, and update products
- Product listing with search capabilities

### 💰 Invoicing System
- Create and manage invoices
- Edit, view, and print invoices
- Download invoices as PDF
- Payment management (mark as paid, view payment details)
- Invoice status management (pending, complete, cancelled)
- Invoice details with line items

### 📄 Quotation Management
- Create and manage quotations
- Edit, view, and download quotations
- Convert quotations to invoices
- Quotation status tracking (draft, accepted, cancelled)

### 📦 Purchase Order Management
- Create and manage purchase orders
- Edit, view, and print purchase orders
- Purchase order status tracking (pending, approved, received, cancelled)
- Supplier information management
- Line item details with product, quantity, and pricing

### 📋 Price Lists & Asset Management
- Create and manage price lists
- Asset management within price lists
- View and edit price list details

### ⚙️ Company Settings
- Configure company information and branding
- Company logo upload management

### 🎨 Responsive & Modern UI
- Clean and professional dashboard interface
- Mobile-responsive design
- Toast notifications for user feedback
- Sidebar navigation with role-based menus

## Tech Stack

- **Backend:** PHP (Native)
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Icons:** Font Awesome
- **Styling:** Custom CSS with responsive design
